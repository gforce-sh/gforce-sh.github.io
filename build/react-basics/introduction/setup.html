Setup

We will need the following:

  • A Github account
  • Git Cli installed
  • Node (at least v18) installed
  • Basic familiarity with Javascript (React is based on this language)
  • An integrated development environment (IDE). Could be Intellij or VScode.

Node​

MacOS
Install using
nvm
. Nvm allows easy switch between node versions.
Alternatively,
install using Brew
Brew is a package manager for MacOS
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
To install Node
brew install node@18
Linux
Install using
nvm-sh
. Nvm-sh allows easy switch between node versions.
Steps
Install nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
The script from the previous step updates the bashrc file. We need to source the updated file:
source ~/.bashrc
Finally, install the node version you'd like:
nvm install v20
To switch between node versions:
nvm use 20
Windows
Install using
nvm-windows
. Nvm-windows allows easy switch between node versions.
Alternatively,
install using Chocolatey
Chocolatey is a package manager for Windows
https://chocolatey.org/install
To install Node
choco install nodejs-lts

To check if all packages have been successfully installed,

git --version
node -v
npm -v
