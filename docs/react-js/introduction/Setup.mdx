# Setup

We will need the following:

- A Github account
- Git Cli installed
- Node (at least v18) installed
- Basic familiarity with Javascript (React is based on this language)
- An integrated development environment (IDE). Could be Intellij or [VScode](https://code.visualstudio.com/download).
